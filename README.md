# Real-Time Character Counter

## Project Description
This project is a simple Character Counter application created using HTML, CSS, and JavaScript. It tracks the number of characters typed inside a textarea in real time.

## Features
- Real-time character counting
- Displays typed characters
- Displays remaining character limit
- Prevents typing beyond maximum limit
- Warning message when limit is reached
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Files Included
- index.html
- style.css
- script.js

## Character Limit
- Maximum 200 characters allowed

## How to Run
1. Download the project files
2. Open the folder in VS Code
3. Open `index.html` in browser

## Output
- Character count updates automatically while typing
- User cannot exceed 200 characters
- Warning message appears when limit is reached
